CSS Basic

 
  
 요소 - 선택자(Selector), 속성(Property), 속성값(Value)
  
 문법(Syntax):
 선택자{속성:속성값;
          속성:속성값;  
 	    ... 
 	 }

CSS 적용방식 3가지

  1. External CSS - 외부파일 방식 *. 로 스타일시트를 따로 분리해서 웹문서의 head태그 내에 link태그로 불러와서 적용 여러 웹문서를 통일성있게 작업가능, 유지보수 용이, 디자인 변경 용이. (가장 추천)
  2. Internal CSS - 웹문서 내장 방식 웹문서의 head태그 내에 style태그(<>로 사용)로 적용 스타일 내용을 적용 (외부파일 방식이랑 병행에서 자주 쯤)
  3. Inline CSS - 태그 뒤 위치방식 웹문서의 body태그 내에 있는 style속성(<>없이 =넣어서 사용)로 스타일 내용을 적용 세가지 방식 중에서 가장 우선순위가 높고, 특수한 상황(블로그 등)에서 쓰임. 유지보수나 통일성 있는 디자인 구성이 힘듦. (가장 비추천하지만 어쩔 수 없이 써야할 상황이 있음)

CSS 주요 선택자 3가지

  1. tag선택자 - tag 그대로 불러와서 적용. 강제적, 자율적 적용.
  2. ID선택자 - tag 뒤에 ID를 지정하고 ID값을 불러와서 적용.
    선택자로 활용할 떄는 ID값 앞에 #을 붙인다.
    ID값은 해당 페이지 내에서 교유하므로 같은 스타일을 다른 요소에 적용 불가. 구조적 용도로
  3. class선택자 - tag 뒤에 class를 지정하고 class 값을 불러와서 적용.
    선택자로 활용할 때는 class값 앞에 .을 붙인다.
    페이지에서 여러번 쓸 수 있으므로 같은 스타일을 다른 요소에 동시에 적용 가능. 주로 디자인 용도로 쓰임.
    (tag 하나에 class값을 2개 이상 조합하여 사용도 가능)
