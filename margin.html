margin의 값의 갯수별 적용법

  1. margin: n ; (바깥 여백의) 양 사방의 값 하나로 동일함
  2. margin: n n ; (바깥 여백의) 상하와 좌우의 순서로 동일함
    영역 전체를 항상 중앙 정렬할 때
    margin: n auto; 로 해서 좌우를 균등분할하면
    요소가 가운데로 오게할 수 있다.
  3. margin: n n n ; (바깥 여백의) 위쪽, 좌우 동일, 아래쪽 순서로 적용
  4. margin: n n n n ; (바깥 여백의) 위쪽, 우측, 아래쪽, 좌측 순서로 적용

margin의 방향별 지정 속성

  1. margin-top: n ; (바깥 여백의) 위쪽
  2. margin-right: n ; (바깥 여백의) 오른쪽
  3. margin-bottom: n ; (바깥 여백의) 아래쪽
  4. margin-left: n ; (바깥 여백의) 왼쪽

box-model의 2가지 타입

box-model은 2가지 타입으로 나뉨 - 줄이 바뀌는 block요소와 - 줄이 바뀌지않는(한줄에 나란히 올 수 있는) inline요소

block요소와 inline요소의 차이점
block 요소 inline 요소
기본 위치 자동으로 줄이 바뀜
(한 줄에 다른 요소와 나란히 올 수 없다)		 줄이 바뀌지 않음
(다른 요소와 나란히 올 수 있다)
